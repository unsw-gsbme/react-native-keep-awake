# react-native-keep-awake

React Native module for keeping the phone screen alive

## Installation

```sh
npm install react-native-keep-awake
```

## Usage

```js
import KeepAwake from "react-native-keep-awake";

// ...

const result = await KeepAwake.multiply(3, 7);
```
